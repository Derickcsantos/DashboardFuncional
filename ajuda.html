Bem-vindo! Estamos aqui para ajudar

Nosso Dashboard vai além de uma simples cópia. Este projeto é uma imersão em tecnologias essenciais, integrando banco de dados local, JavaScript avançado, Css dinamico e responsivo, técnicas de debugging, manipulação e experiencia com C.R.U.D e interação com diversas APIs. Ele foi projetado para não só exibir dados, mas também oferecer uma experiência dinâmica e enriquecedora para o usuario, além de experiencia e portfolio.

FAQ

Perguntas frequentes.

Para atualizar dados em sistema, basta acessar o icone de usuario na parte superior direito na pagina principal. Após isto atualize os dados que deseja e clique no botão "Atualizar"

Para consultar os dados dos usuarios cadastrados em sistema, basta acessar o menu lateral e clicar no icone de usuarios, após isto, clique em carregar dados. Lá será possivel visualizar "ID", "Nome", "Telefone", "E-mail" e até "Senha" dos usuarios cadastrados. Além disso conta com a opção de excluir caso queira realizar uma limpa em sistema

Para atualizar os valores financeiros e quantidade de funcionarios/clientes, basta acessar o serviço "Ganhos" no menu lateral localizado na area lateral esquerda da pagina principal. Lá será possivel definir valores dos ganhos, além de poder definir a quantidade de clientes e o numero de funcionarios.
